Zapstore is an open Android app store built on Nostr. Apps are signed by developers, verified before install, and discoverable through your social graph.

**What you need:** An Android device (10+, arm64).

## 1. Install Zapstore

Download the APK from [zapstore.dev](/) and open it. Android will ask you to allow installs from this source — tap **Allow**.

Once installed, open Zapstore. You don't need a Nostr account to browse and install apps.

## 2. Browse or search

- **Discover** tab — curated apps and stacks
- **Apps** tab — full catalog, sorted by recent activity
- **Search** — type an app name or paste a GitHub/GitLab URL to find a specific app

## 3. Install an app

Tap any app, then tap **Install**. Zapstore fetches the release, verifies the developer's cryptographic signature, and hands off to Android's package installer.

If the signature check fails, the install is blocked and you're told why.

## Connect your Nostr identity (optional)

Sign in with a Nostr key or browser extension to:

- See which apps people you follow use
- Send Bitcoin directly to developers via Lightning zaps
- Leave comments and reviews
